Class folder playlist

reduce

Description

a folder that contains other playlists

  • Plural name:
    • folder playlists
  • Defined in:
    • iTunes Suite
  • Inherits from:
    • user playlist (iTunes Suite) → playlist (iTunes Suite) → item (iTunes Suite)

Properties

  • container (r/o) reference -- the container of the item
  • id (r/o) integer -- the id of the item
  • index (r/o) integer -- The index of the item in internal application order.
  • persistent ID (r/o) string -- the id of the item as a hexidecimal string. This id does not change over time.
  • duration (r/o) integer -- the total length of all songs (in seconds)
  • name Unicode text -- the name of the playlist
  • parent (r/o) playlist -- folder which contains this playlist (if any)
  • shuffle boolean -- play the songs in this playlist in random order?
  • size (r/o) double integer -- the total size of all songs (in bytes)
  • song repeat off / one / all -- playback repeat mode
  • special kind (r/o) none / Books / folder / Genius / iTunes U / Library / Movies / Music / Party Shuffle / Podcasts / Purchased Music / TV Shows -- special playlist kind
  • time (r/o) Unicode text -- the length of all songs in MM:SS format
  • visible (r/o) boolean -- is this playlist visible in the Source list?
  • shared boolean -- is this playlist shared?
  • smart (r/o) boolean -- is this a Smart Playlist?

Elements

  • track -- by index, name, id
  • file track -- by index, name, id
  • URL track -- by index, name, id
  • shared track -- by index, name, id, range, relative, test
