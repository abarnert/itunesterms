Class artwork

reduce

Description

a piece of art within a track

  • Plural name:
    • artworks
  • Defined in:
    • iTunes Suite
  • Inherits from:
    • item (iTunes Suite)

Properties

  • container (r/o) reference -- the container of the item
  • id (r/o) integer -- the id of the item
  • index (r/o) integer -- The index of the item in internal application order.
  • name Unicode text -- the name of the item
  • persistent ID (r/o) string -- the id of the item as a hexidecimal string. This id does not change over time.
  • data picture -- data for this artwork, in the form of a picture
  • description Unicode text -- description of artwork as a string
  • downloaded (r/o) boolean -- was this artwork downloaded by iTunes?
  • format (r/o) type class -- the data format for this piece of artwork
  • kind integer -- kind or purpose of this piece of artwork
  • raw data <class tdta> -- data for this artwork, in original format

Notes

  • Contained by:
    • track (iTunes Suite)
