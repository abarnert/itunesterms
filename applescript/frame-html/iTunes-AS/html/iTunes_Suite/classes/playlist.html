Class playlist

expand

Description

a list of songs/streams

  • Plural name:
    • playlists
  • Defined in:
    • iTunes Suite
  • Inherits from:
    • item (iTunes Suite)

Properties

  • duration (r/o) integer -- the total length of all songs (in seconds)
  • name Unicode text -- the name of the playlist
  • parent (r/o) playlist -- folder which contains this playlist (if any)
  • shuffle boolean -- play the songs in this playlist in random order?
  • size (r/o) double integer -- the total size of all songs (in bytes)
  • song repeat off / one / all -- playback repeat mode
  • special kind (r/o) none / Books / folder / Genius / iTunes U / Library / Movies / Music / Party Shuffle / Podcasts / Purchased Music / TV Shows -- special playlist kind
  • time (r/o) Unicode text -- the length of all songs in MM:SS format
  • visible (r/o) boolean -- is this playlist visible in the Source list?

Elements

  • track -- by index, name, id
