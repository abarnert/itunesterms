iTunes.app terminology

/Applications/iTunes.app

Standard Suite

Common terms for most applications

Commands

  • print -- Print the specified object(s)
    • [reference] -- list of objects to print
    • [print dialog boolean] -- Should the application show the print dialog
    • [with properties print settings] -- the print settings
    • [kind track listing / album listing / cd insert] -- the kind of printout desired
    • [theme Unicode text] -- name of theme to use for formatting the printout
  • close -- Close an object
    • reference -- the object to close
  • count -- Return the number of elements of a particular class within an object
    • reference -- the object whose elements are to be counted
    • each type class -- the class of the elements to be counted. Keyword 'each' is optional in AppleScript
    • Result: integer -- the number of elements
  • delete -- Delete an element from an object
    • reference -- the element to delete
  • duplicate -- Duplicate one or more object(s)
    • reference -- the object(s) to duplicate
    • [to location reference] -- the new location for the object(s)
    • Result: reference -- to the duplicated object(s)
  • exists -- Verify if an object exists
    • reference -- the object in question
    • Result: boolean -- true if it exists, false if not
  • make -- Make a new element
    • new type class -- the class of the new element. Keyword 'new' is optional in AppleScript
    • [at location reference] -- the location at which to insert the element
    • [with properties record] -- the initial values for the properties of the element
    • Result: reference -- to the new object(s)
  • move -- Move playlist(s) to a new location
    • playlist -- the playlist(s) to move
    • to location reference -- the new location for the playlist(s)
  • open -- open the specified object(s)
    • reference -- list of objects to open
  • run -- run iTunes
  • quit -- quit iTunes
  • set
    • reference
    • to anything

Classes

  • print settings
    • Properties:
      • copies (r/o) integer -- the number of copies of a document to be printed
      • collating (r/o) boolean -- Should printed copies be collated?
      • starting page (r/o) integer -- the first page of the document to be printed
      • ending page (r/o) integer -- the last page of the document to be printed
      • pages across (r/o) integer -- number of logical pages laid across a physical page
      • pages down (r/o) integer -- number of logical pages laid out down a physical page
      • error handling (r/o) standard / detailed -- how errors are handled
      • requested print time (r/o) date -- the time at which the desktop printer should print the document
      • printer features (r/o) list -- printer specific options
      • fax number (r/o) string -- for fax number
      • target printer (r/o) string -- for target printer

iTunes Suite

The event suite specific to iTunes

Commands

  • add -- add one or more files to a playlist
    • list of alias -- the file(s) to add
    • [to location reference] -- the location of the added file(s)
    • Result: track -- reference to added track(s)
  • back track -- reposition to beginning of current track or go to previous track if already at start of current track
  • convert -- convert one or more files or tracks
    • list of reference -- the file(s)/tracks(s) to convert
    • Result: track -- reference to converted track(s)
  • fast forward -- skip forward in a playing track
  • next track -- advance to the next track in the current playlist
  • pause -- pause playback
  • play -- play the current track or the specified track or file.
    • [reference] -- item to play
    • [once boolean] -- If true, play this track once and then stop.
  • playpause -- toggle the playing/paused state of the current track
  • previous track -- return to the previous track in the current playlist
  • refresh -- update file track information from the current information in the track’s file
    • file track -- the file track to update
  • resume -- disable fast forward/rewind and resume playback, if playing.
  • reveal -- reveal and select a track or playlist
    • item -- the item to reveal
  • rewind -- skip backwards in a playing track
  • search -- search a playlist for tracks matching the search string. Identical to entering search text in the Search field in iTunes.
    • playlist -- the playlist to search
    • for Unicode text -- the search text
    • [only albums / all / artists / composers / displayed / songs] -- area to search (default is all)
    • Result: track -- reference to found track(s)
  • stop -- stop playback
  • update -- update the specified iPod
    • [source] -- the iPod to update
  • eject -- eject the specified iPod
    • [source] -- the iPod to eject
  • subscribe -- subscribe to a podcast feed
    • string -- the URL of the feed to subscribe to
  • updateAllPodcasts -- update all subscribed podcast feeds
  • updatePodcast -- update podcast feed
  • download -- download podcast episode
    • URL track

Classes

  • application -- The application program
    • Properties:
      • current encoder encoder -- the currently selected encoder (MP3, AIFF, WAV, etc.)
      • current EQ preset EQ preset -- the currently selected equalizer preset
      • current playlist (r/o) playlist -- the playlist containing the currently targeted track
      • current stream title (r/o) Unicode text -- the name of the current song in the playing stream (provided by streaming server)
      • current stream URL (r/o) Unicode text -- the URL of the playing stream or streaming web site (provided by streaming server)
      • current track (r/o) track -- the current targeted track
      • current visual visual -- the currently selected visual plug-in
      • EQ enabled boolean -- is the equalizer enabled?
      • fixed indexing boolean -- true if all AppleScript track indices should be independent of the play order of the owning playlist.
      • frontmost boolean -- is iTunes the frontmost application?
      • full screen boolean -- are visuals displayed using the entire screen?
      • name (r/o) Unicode text -- the name of the application
      • mute boolean -- has the sound output been muted?
      • player position integer -- the player’s position within the currently playing track in seconds.
      • player state (r/o) stopped / playing / paused / fast forwarding / rewinding -- is iTunes stopped, paused, or playing?
      • selection (r/o) reference -- the selection visible to the user
      • sound volume integer -- the sound output volume (0 = minimum, 100 = maximum)
      • version (r/o) version -- the version of iTunes
      • visuals enabled boolean -- are visuals currently being displayed?
      • visual size small / medium / large -- the size of the displayed visual
    • Elements:
      • browser window -- by index, name, id
      • encoder -- by index, name, id
      • EQ preset -- by index, name, id
      • EQ window -- by index, name, id
      • playlist window -- by index, name, id
      • source -- by index, name, id
      • visual -- by index, name, id
      • window -- by index, name, id
  • artwork -- a piece of art within a track
    • Plural name:
      • artworks
    • Inherits from:
      • item
    • Properties:
      • data picture -- data for this artwork, in the form of a picture
      • description Unicode text -- description of artwork as a string
      • downloaded (r/o) boolean -- was this artwork downloaded by iTunes?
      • format (r/o) type class -- the data format for this piece of artwork
      • kind integer -- kind or purpose of this piece of artwork
      • raw data <class tdta> -- data for this artwork, in original format
  • audio CD playlist -- a playlist representing an audio CD
    • Plural name:
      • audio CD playlists
    • Inherits from:
      • playlist
    • Properties:
      • artist Unicode text -- the artist of the CD
      • compilation boolean -- is this CD a compilation album?
      • composer Unicode text -- the composer of the CD
      • disc count integer -- the total number of discs in this CD’s album
      • disc number integer -- the index of this CD disc in the source album
      • genre Unicode text -- the genre of the CD
      • year integer -- the year the album was recorded/released
    • Elements:
      • audio CD track -- by index, name, id
  • audio CD track -- a track on an audio CD
    • Plural name:
      • audio CD tracks
    • Inherits from:
      • track
    • Properties:
      • location (r/o) alias -- the location of the file represented by this track
  • browser window -- the main iTunes window
    • Plural name:
      • browser windows
    • Inherits from:
      • window
    • Properties:
      • minimized boolean -- is the small player visible?
      • selection (r/o) reference -- the selected songs
      • view playlist -- the playlist currently displayed in the window
  • device playlist -- a playlist representing the contents of a portable device
    • Plural name:
      • device playlists
    • Inherits from:
      • playlist
    • Elements:
      • device track -- by index, name, id
  • device track -- a track residing on a portable music player
    • Plural name:
      • device tracks
    • Inherits from:
      • track
  • encoder -- converts a track to a specific file format
    • Plural name:
      • encoders
    • Inherits from:
      • item
    • Properties:
      • format (r/o) Unicode text -- the data format created by the encoder
  • EQ preset -- equalizer preset configuration
    • Plural name:
      • EQ presets
    • Inherits from:
      • item
    • Properties:
      • band 1 small real -- the equalizer 32 Hz band level (-12.0 dB to +12.0 dB)
      • band 2 small real -- the equalizer 64 Hz band level (-12.0 dB to +12.0 dB)
      • band 3 small real -- the equalizer 125 Hz band level (-12.0 dB to +12.0 dB)
      • band 4 small real -- the equalizer 250 Hz band level (-12.0 dB to +12.0 dB)
      • band 5 small real -- the equalizer 500 Hz band level (-12.0 dB to +12.0 dB)
      • band 6 small real -- the equalizer 1 kHz band level (-12.0 dB to +12.0 dB)
      • band 7 small real -- the equalizer 2 kHz band level (-12.0 dB to +12.0 dB)
      • band 8 small real -- the equalizer 4 kHz band level (-12.0 dB to +12.0 dB)
      • band 9 small real -- the equalizer 8 kHz band level (-12.0 dB to +12.0 dB)
      • band 10 small real -- the equalizer 16 kHz band level (-12.0 dB to +12.0 dB)
      • modifiable (r/o) boolean -- can this preset be modified?
      • preamp small real -- the equalizer preamp level (-12.0 dB to +12.0 dB)
      • update tracks boolean -- should tracks which refer to this preset be updated when the preset is renamed or deleted?
  • EQ window -- the iTunes equalizer window
    • Plural name:
      • EQ windows
    • Inherits from:
      • window
    • Properties:
      • minimized boolean -- is the small EQ window visible?
  • file track -- a track representing an audio file (MP3, AIFF, etc.)
    • Plural name:
      • file tracks
    • Inherits from:
      • track
    • Properties:
      • location alias -- the location of the file represented by this track
  • folder playlist -- a folder that contains other playlists
    • Plural name:
      • folder playlists
    • Inherits from:
      • user playlist
  • item -- an item
    • Plural name:
      • items
    • Inherited by:
      • artwork
      • encoder
      • EQ preset
      • playlist
      • source
      • track
      • visual
      • window
    • Properties:
      • container (r/o) reference -- the container of the item
      • id (r/o) integer -- the id of the item
      • index (r/o) integer -- The index of the item in internal application order.
      • name Unicode text -- the name of the item
      • persistent ID (r/o) string -- the id of the item as a hexidecimal string. This id does not change over time.
  • library playlist -- the master music library playlist
    • Plural name:
      • library playlists
    • Inherits from:
      • playlist
    • Elements:
      • file track -- by index, name, id
      • URL track -- by index, name, id
      • shared track -- by index, name, id, range, relative, test
  • playlist -- a list of songs/streams
    • Plural name:
      • playlists
    • Inherits from:
      • item
    • Inherited by:
      • audio CD playlist
      • device playlist
      • library playlist
      • radio tuner playlist
      • user playlist
    • Properties:
      • duration (r/o) integer -- the total length of all songs (in seconds)
      • name Unicode text -- the name of the playlist
      • parent (r/o) playlist -- folder which contains this playlist (if any)
      • shuffle boolean -- play the songs in this playlist in random order?
      • size (r/o) double integer -- the total size of all songs (in bytes)
      • song repeat off / one / all -- playback repeat mode
      • special kind (r/o) none / Books / folder / Genius / iTunes U / Library / Movies / Music / Party Shuffle / Podcasts / Purchased Music / TV Shows -- special playlist kind
      • time (r/o) Unicode text -- the length of all songs in MM:SS format
      • visible (r/o) boolean -- is this playlist visible in the Source list?
    • Elements:
      • track -- by index, name, id
  • playlist window -- a sub-window showing a single playlist
    • Plural name:
      • playlist windows
    • Inherits from:
      • window
    • Properties:
      • selection (r/o) reference -- the selected songs
      • view (r/o) playlist -- the playlist displayed in the window
  • radio tuner playlist -- the radio tuner playlist
    • Plural name:
      • radio tuner playlists
    • Inherits from:
      • playlist
    • Elements:
      • URL track -- by index, name, id
  • shared track -- a track residing in a shared library
    • Plural name:
      • shared tracks
    • Inherits from:
      • track
  • source -- a music source (music library, CD, device, etc.)
    • Plural name:
      • sources
    • Inherits from:
      • item
    • Properties:
      • capacity (r/o) double integer -- the total size of the source if it has a fixed size
      • free space (r/o) double integer -- the free space on the source if it has a fixed size
      • kind (r/o) library / iPod / audio CD / MP3 CD / device / radio tuner / shared library / unknown
    • Elements:
      • audio CD playlist -- by index, name, id
      • device playlist -- by index, name, id
      • library playlist -- by index, name, id
      • playlist -- by index, name, id
      • radio tuner playlist -- by index, name, id
      • user playlist -- by index, name, id
  • track -- playable audio source
    • Plural name:
      • tracks
    • Inherits from:
      • item
    • Inherited by:
      • audio CD track
      • device track
      • file track
      • shared track
      • URL track
    • Properties:
      • album Unicode text -- the album name of the track
      • album artist Unicode text -- the album artist of the track
      • album rating integer -- the rating of the album for this track (0 to 100)
      • album rating kind (r/o) user / computed -- the rating kind of the album rating for this track
      • artist Unicode text -- the artist/source of the track
      • bit rate (r/o) integer -- the bit rate of the track (in kbps)
      • bookmark small real -- the bookmark time of the track in seconds
      • bookmarkable boolean -- is the playback position for this track remembered?
      • bpm integer -- the tempo of this track in beats per minute
      • category Unicode text -- the category of the track
      • comment Unicode text -- freeform notes about the track
      • compilation boolean -- is this track from a compilation album?
      • composer Unicode text -- the composer of the track
      • database ID (r/o) integer -- the common, unique ID for this track. If two tracks in different playlists have the same database ID, they are sharing the same data.
      • date added (r/o) date -- the date the track was added to the playlist
      • description Unicode text -- the description of the track
      • disc count integer -- the total number of discs in the source album
      • disc number integer -- the index of the disc containing this track on the source album
      • duration (r/o) small real -- the length of the track in seconds
      • enabled boolean -- is this track checked for playback?
      • episode ID Unicode text -- the episode ID of the track
      • episode number integer -- the episode number of the track
      • EQ Unicode text -- the name of the EQ preset of the track
      • finish small real -- the stop time of the track in seconds
      • gapless boolean -- is this track from a gapless album?
      • genre Unicode text -- the music/audio genre (category) of the track
      • grouping Unicode text -- the grouping (piece) of the track. Generally used to denote movements within a classical work.
      • kind (r/o) Unicode text -- a text description of the track
      • long description Unicode text
      • lyrics Unicode text -- the lyrics of the track
      • modification date (r/o) date -- the modification date of the content of this track
      • played count integer -- number of times this track has been played
      • played date date -- the date and time this track was last played
      • podcast (r/o) boolean -- is this track a podcast episode?
      • rating integer -- the rating of this track (0 to 100)
      • rating kind (r/o) user / computed -- the rating kind of this track
      • release date (r/o) date -- the release date of this track
      • sample rate (r/o) integer -- the sample rate of the track (in Hz)
      • season number integer -- the season number of the track
      • shufflable boolean -- is this track included when shuffling?
      • skipped count integer -- number of times this track has been skipped
      • skipped date date -- the date and time this track was last skipped
      • show Unicode text -- the show name of the track
      • sort album Unicode text -- override string to use for the track when sorting by album
      • sort artist Unicode text -- override string to use for the track when sorting by artist
      • sort album artist Unicode text -- override string to use for the track when sorting by album artist
      • sort name Unicode text -- override string to use for the track when sorting by name
      • sort composer Unicode text -- override string to use for the track when sorting by composer
      • sort show Unicode text -- override string to use for the track when sorting by show name
      • size (r/o) integer -- the size of the track (in bytes)
      • start small real -- the start time of the track in seconds
      • time (r/o) Unicode text -- the length of the track in MM:SS format
      • track count integer -- the total number of tracks on the source album
      • track number integer -- the index of the track on the source album
      • unplayed boolean -- is this track unplayed?
      • video kind none / movie / music video / TV show -- kind of video track
      • volume adjustment integer -- relative volume adjustment of the track (-100% to 100%)
      • year integer -- the year the track was recorded/released
    • Elements:
      • artwork -- by index
  • URL track -- a track representing a network stream
    • Plural name:
      • URL tracks
    • Inherits from:
      • track
    • Properties:
      • address Unicode text -- the URL for this track
  • user playlist -- custom playlists created by the user
    • Plural name:
      • user playlists
    • Inherits from:
      • playlist
    • Inherited by:
      • folder playlist
    • Properties:
      • shared boolean -- is this playlist shared?
      • smart (r/o) boolean -- is this a Smart Playlist?
    • Elements:
      • file track -- by index, name, id
      • URL track -- by index, name, id
      • shared track -- by index, name, id, range, relative, test
  • visual -- a visual plug-in
    • Plural name:
      • visuals
    • Inherits from:
      • item
  • window -- any window
    • Plural name:
      • windows
    • Inherits from:
      • item
    • Inherited by:
      • browser window
      • EQ window
      • playlist window
    • Properties:
      • bounds bounding rectangle -- the boundary rectangle for the window
      • closeable (r/o) boolean -- does the window have a close box?
      • collapseable (r/o) boolean -- does the window have a collapse (windowshade) box?
      • collapsed boolean -- is the window collapsed?
      • position point -- the upper left position of the window
      • resizable (r/o) boolean -- is the window resizable?
      • visible boolean -- is the window visible?
      • zoomable (r/o) boolean -- is the window zoomable?
      • zoomed boolean -- is the window zoomed?
