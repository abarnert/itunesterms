Class folder_playlist

reduce

Description

a folder that contains other playlists

  • Plural name:
    • folder_playlists
  • Defined in:
    • iTunes Suite
  • Inherits from:
    • user_playlist (iTunes Suite) → playlist (iTunes Suite) → item (iTunes Suite)

Properties

  • container (r/o) reference -- the container of the item
  • id (r/o) integer -- the id of the item
  • index (r/o) integer -- The index of the item in internal application order.
  • persistent_ID (r/o) string -- the id of the item as a hexidecimal string. This id does not change over time.
  • duration (r/o) integer -- the total length of all songs (in seconds)
  • name unicode_text -- the name of the playlist
  • parent (r/o) playlist -- folder which contains this playlist (if any)
  • shuffle boolean -- play the songs in this playlist in random order?
  • size (r/o) double_integer -- the total size of all songs (in bytes)
  • song_repeat k.off / k.one / k.all -- playback repeat mode
  • special_kind (r/o) k.none / k.Books / k.folder / k.Genius / k.iTunes_U / k.Library / k.Movies / k.Music / k.Party_Shuffle / k.Podcasts / k.Purchased_Music / k.TV_Shows -- special playlist kind
  • time (r/o) unicode_text -- the length of all songs in MM:SS format
  • visible (r/o) boolean -- is this playlist visible in the Source list?
  • shared boolean -- is this playlist shared?
  • smart (r/o) boolean -- is this a Smart Playlist?

Elements

  • tracks -- by index, name, id
  • file_tracks -- by index, name, id
  • URL_tracks -- by index, name, id
  • shared_tracks -- by index, name, id, range, relative, test
