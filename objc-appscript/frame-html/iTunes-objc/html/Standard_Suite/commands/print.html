Command print

Description

Print the specified object(s)

  • Defined in:
    • Standard Suite

Parameters

  • [reference] -- list of objects to print
  • [printDialog boolean] -- Should the application show the print dialog
  • [withProperties printSettings] -- the print settings
  • [kind trackListing / albumListing / cdInsert] -- the kind of printout desired
  • [theme unicodeText] -- name of theme to use for formatting the printout
