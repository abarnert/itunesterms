Class printSettings

expand

Description

  • Defined in:
    • Standard Suite

Properties

  • copies (r/o) integer -- the number of copies of a document to be printed
  • collating (r/o) boolean -- Should printed copies be collated?
  • startingPage (r/o) integer -- the first page of the document to be printed
  • endingPage (r/o) integer -- the last page of the document to be printed
  • pagesAcross (r/o) integer -- number of logical pages laid across a physical page
  • pagesDown (r/o) integer -- number of logical pages laid out down a physical page
  • errorHandling (r/o) standard / detailed -- how errors are handled
  • requestedPrintTime (r/o) date -- the time at which the desktop printer should print the document
  • printerFeatures (r/o) list -- printer specific options
  • faxNumber (r/o) string -- for fax number
  • targetPrinter (r/o) string -- for target printer
