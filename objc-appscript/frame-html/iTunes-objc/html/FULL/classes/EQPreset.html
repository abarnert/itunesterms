Class EQPreset

reduce

Description

equalizer preset configuration

  • Plural name:
    • EQPresets
  • Defined in:
    • iTunes Suite
  • Inherits from:
    • item (iTunes Suite)

Properties

  • container (r/o) reference -- the container of the item
  • id_ (r/o) integer -- the id of the item
  • index (r/o) integer -- The index of the item in internal application order.
  • name unicodeText -- the name of the item
  • persistentID (r/o) string -- the id of the item as a hexidecimal string. This id does not change over time.
  • band1 shortFloat -- the equalizer 32 Hz band level (-12.0 dB to +12.0 dB)
  • band2 shortFloat -- the equalizer 64 Hz band level (-12.0 dB to +12.0 dB)
  • band3 shortFloat -- the equalizer 125 Hz band level (-12.0 dB to +12.0 dB)
  • band4 shortFloat -- the equalizer 250 Hz band level (-12.0 dB to +12.0 dB)
  • band5 shortFloat -- the equalizer 500 Hz band level (-12.0 dB to +12.0 dB)
  • band6 shortFloat -- the equalizer 1 kHz band level (-12.0 dB to +12.0 dB)
  • band7 shortFloat -- the equalizer 2 kHz band level (-12.0 dB to +12.0 dB)
  • band8 shortFloat -- the equalizer 4 kHz band level (-12.0 dB to +12.0 dB)
  • band9 shortFloat -- the equalizer 8 kHz band level (-12.0 dB to +12.0 dB)
  • band10 shortFloat -- the equalizer 16 kHz band level (-12.0 dB to +12.0 dB)
  • modifiable (r/o) boolean -- can this preset be modified?
  • preamp shortFloat -- the equalizer preamp level (-12.0 dB to +12.0 dB)
  • updateTracks boolean -- should tracks which refer to this preset be updated when the preset is renamed or deleted?

Notes

  • Contained by:
    • application (iTunes Suite)
