Class source

reduce

Description

a music source (music library, CD, device, etc.)

  • Plural name:
    • sources
  • Defined in:
    • iTunes Suite
  • Inherits from:
    • item (iTunes Suite)

Properties

  • container (r/o) reference -- the container of the item
  • id_ (r/o) integer -- the id of the item
  • index (r/o) integer -- The index of the item in internal application order.
  • name unicodeText -- the name of the item
  • persistentID (r/o) string -- the id of the item as a hexidecimal string. This id does not change over time.
  • capacity (r/o) doubleInteger -- the total size of the source if it has a fixed size
  • freeSpace (r/o) doubleInteger -- the free space on the source if it has a fixed size
  • kind (r/o) library / iPod / audioCD / MP3CD / device / radioTuner / sharedLibrary / unknown

Elements

  • audioCDPlaylists -- by index, name, id
  • devicePlaylists -- by index, name, id
  • libraryPlaylists -- by index, name, id
  • playlists -- by index, name, id
  • radioTunerPlaylists -- by index, name, id
  • userPlaylists -- by index, name, id

Notes

  • Contained by:
    • application (iTunes Suite)
