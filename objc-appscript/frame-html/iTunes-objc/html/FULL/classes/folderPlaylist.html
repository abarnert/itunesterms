Class folderPlaylist

reduce

Description

a folder that contains other playlists

  • Plural name:
    • folderPlaylists
  • Defined in:
    • iTunes Suite
  • Inherits from:
    • userPlaylist (iTunes Suite) → playlist (iTunes Suite) → item (iTunes Suite)

Properties

  • container (r/o) reference -- the container of the item
  • id_ (r/o) integer -- the id of the item
  • index (r/o) integer -- The index of the item in internal application order.
  • persistentID (r/o) string -- the id of the item as a hexidecimal string. This id does not change over time.
  • duration (r/o) integer -- the total length of all songs (in seconds)
  • name unicodeText -- the name of the playlist
  • parent (r/o) playlist -- folder which contains this playlist (if any)
  • shuffle boolean -- play the songs in this playlist in random order?
  • size (r/o) doubleInteger -- the total size of all songs (in bytes)
  • songRepeat off / one / all -- playback repeat mode
  • specialKind (r/o) none / Books / folder / Genius / iTunesU / Library / Movies / Music / PartyShuffle / Podcasts / PurchasedMusic / TVShows -- special playlist kind
  • time (r/o) unicodeText -- the length of all songs in MM:SS format
  • visible (r/o) boolean -- is this playlist visible in the Source list?
  • shared boolean -- is this playlist shared?
  • smart (r/o) boolean -- is this a Smart Playlist?

Elements

  • tracks -- by index, name, id
  • fileTracks -- by index, name, id
  • URLTracks -- by index, name, id
  • sharedTracks -- by index, name, range, relative, test
