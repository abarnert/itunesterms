Class playlist

expand

Description

a list of songs/streams

  • Plural name:
    • playlists
  • Defined in:
    • iTunes Suite
  • Inherits from:
    • item (iTunes Suite)

Properties

  • duration (r/o) integer -- the total length of all songs (in seconds)
  • name unicodeText -- the name of the playlist
  • parent (r/o) playlist -- folder which contains this playlist (if any)
  • shuffle boolean -- play the songs in this playlist in random order?
  • size (r/o) doubleInteger -- the total size of all songs (in bytes)
  • songRepeat off / one / all -- playback repeat mode
  • specialKind (r/o) none / Books / folder / Genius / iTunesU / Library / Movies / Music / PartyShuffle / Podcasts / PurchasedMusic / TVShows -- special playlist kind
  • time (r/o) unicodeText -- the length of all songs in MM:SS format
  • visible (r/o) boolean -- is this playlist visible in the Source list?

Elements

  • tracks -- by index, name, id
