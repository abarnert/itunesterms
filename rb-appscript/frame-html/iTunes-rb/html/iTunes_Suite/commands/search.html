Command search

Description

search a playlist for tracks matching the search string. Identical to entering search text in the Search field in iTunes.

  • Defined in:
    • iTunes Suite

Parameters

  • playlist -- the playlist to search
  • for unicode_text -- the search text
  • [only :albums / :all / :artists / :composers / :displayed / :songs] -- area to search (default is all)

Result

  • track -- reference to found track(s)
