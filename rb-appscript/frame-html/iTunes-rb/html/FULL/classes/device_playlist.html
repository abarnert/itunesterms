Class device_playlist

reduce

Description

a playlist representing the contents of a portable device

  • Plural name:
    • device_playlists
  • Defined in:
    • iTunes Suite
  • Inherits from:
    • playlist (iTunes Suite) → item (iTunes Suite)

Properties

  • container (r/o) reference -- the container of the item
  • id_ (r/o) integer -- the id of the item
  • index (r/o) integer -- The index of the item in internal application order.
  • persistent_ID (r/o) string -- the id of the item as a hexidecimal string. This id does not change over time.
  • duration (r/o) integer -- the total length of all songs (in seconds)
  • name unicode_text -- the name of the playlist
  • parent (r/o) playlist -- folder which contains this playlist (if any)
  • shuffle boolean -- play the songs in this playlist in random order?
  • size (r/o) double_integer -- the total size of all songs (in bytes)
  • song_repeat :off / :one / :all -- playback repeat mode
  • special_kind (r/o) :none / :Books / :folder / :Genius / :iTunes_U / :Library / :Movies / :Music / :Party_Shuffle / :Podcasts / :Purchased_Music / :TV_Shows -- special playlist kind
  • time (r/o) unicode_text -- the length of all songs in MM:SS format
  • visible (r/o) boolean -- is this playlist visible in the Source list?

Elements

  • tracks -- by index, name, id
  • device_tracks -- by index, name, id

Notes

  • Contained by:
    • source (iTunes Suite)
