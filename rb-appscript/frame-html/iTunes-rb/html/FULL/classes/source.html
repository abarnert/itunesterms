Class source

reduce

Description

a music source (music library, CD, device, etc.)

  • Plural name:
    • sources
  • Defined in:
    • iTunes Suite
  • Inherits from:
    • item (iTunes Suite)

Properties

  • container (r/o) reference -- the container of the item
  • id_ (r/o) integer -- the id of the item
  • index (r/o) integer -- The index of the item in internal application order.
  • name unicode_text -- the name of the item
  • persistent_ID (r/o) string -- the id of the item as a hexidecimal string. This id does not change over time.
  • capacity (r/o) double_integer -- the total size of the source if it has a fixed size
  • free_space (r/o) double_integer -- the free space on the source if it has a fixed size
  • kind (r/o) :library / :iPod / :audio_CD / :MP3_CD / :device / :radio_tuner / :shared_library / :unknown

Elements

  • audio_CD_playlists -- by index, name, id
  • device_playlists -- by index, name, id
  • library_playlists -- by index, name, id
  • playlists -- by index, name, id
  • radio_tuner_playlists -- by index, name, id
  • user_playlists -- by index, name, id

Notes

  • Contained by:
    • application (iTunes Suite)
