Class print_settings

reduce

Description

  • Defined in:
    • Standard Suite

Properties

  • copies (r/o) integer -- the number of copies of a document to be printed
  • collating (r/o) boolean -- Should printed copies be collated?
  • starting_page (r/o) integer -- the first page of the document to be printed
  • ending_page (r/o) integer -- the last page of the document to be printed
  • pages_across (r/o) integer -- number of logical pages laid across a physical page
  • pages_down (r/o) integer -- number of logical pages laid out down a physical page
  • error_handling (r/o) :standard / :detailed -- how errors are handled
  • requested_print_time (r/o) date -- the time at which the desktop printer should print the document
  • printer_features (r/o) list -- printer specific options
  • fax_number (r/o) string -- for fax number
  • target_printer (r/o) string -- for target printer
