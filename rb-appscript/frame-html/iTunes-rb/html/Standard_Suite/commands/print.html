Command print

Description

Print the specified object(s)

  • Defined in:
    • Standard Suite

Parameters

  • [reference] -- list of objects to print
  • [print_dialog boolean] -- Should the application show the print dialog
  • [with_properties print_settings] -- the print settings
  • [kind :track_listing / :album_listing / :cd_insert] -- the kind of printout desired
  • [theme unicode_text] -- name of theme to use for formatting the printout
