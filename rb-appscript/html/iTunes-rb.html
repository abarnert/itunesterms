iTunes.app terminology

/Applications/iTunes.app

Standard Suite

Common terms for most applications

Commands

  • print -- Print the specified object(s)
    • [reference] -- list of objects to print
    • [print_dialog boolean] -- Should the application show the print dialog
    • [with_properties print_settings] -- the print settings
    • [kind :track_listing / :album_listing / :cd_insert] -- the kind of printout desired
    • [theme unicode_text] -- name of theme to use for formatting the printout
  • close -- Close an object
    • reference -- the object to close
  • count -- Return the number of elements of a particular class within an object
    • reference -- the object whose elements are to be counted
    • each type_class -- the class of the elements to be counted. Keyword 'each' is optional in AppleScript
    • Result: integer -- the number of elements
  • delete -- Delete an element from an object
    • reference -- the element to delete
  • duplicate -- Duplicate one or more object(s)
    • reference -- the object(s) to duplicate
    • [to location_reference] -- the new location for the object(s)
    • Result: reference -- to the duplicated object(s)
  • exists -- Verify if an object exists
    • reference -- the object in question
    • Result: boolean -- true if it exists, false if not
  • make -- Make a new element
    • new type_class -- the class of the new element. Keyword 'new' is optional in AppleScript
    • [at location_reference] -- the location at which to insert the element
    • [with_properties record] -- the initial values for the properties of the element
    • Result: reference -- to the new object(s)
  • move -- Move playlist(s) to a new location
    • playlist -- the playlist(s) to move
    • to location_reference -- the new location for the playlist(s)
  • open -- open the specified object(s)
    • reference -- list of objects to open
  • run -- run iTunes
  • quit -- quit iTunes
  • set
    • reference
    • to anything

Classes

  • print_settings
    • Properties:
      • copies (r/o) integer -- the number of copies of a document to be printed
      • collating (r/o) boolean -- Should printed copies be collated?
      • starting_page (r/o) integer -- the first page of the document to be printed
      • ending_page (r/o) integer -- the last page of the document to be printed
      • pages_across (r/o) integer -- number of logical pages laid across a physical page
      • pages_down (r/o) integer -- number of logical pages laid out down a physical page
      • error_handling (r/o) :standard / :detailed -- how errors are handled
      • requested_print_time (r/o) date -- the time at which the desktop printer should print the document
      • printer_features (r/o) list -- printer specific options
      • fax_number (r/o) string -- for fax number
      • target_printer (r/o) string -- for target printer

iTunes Suite

The event suite specific to iTunes

Commands

  • add -- add one or more files to a playlist
    • list of alias -- the file(s) to add
    • [to location_reference] -- the location of the added file(s)
    • Result: track -- reference to added track(s)
  • back_track -- reposition to beginning of current track or go to previous track if already at start of current track
  • convert -- convert one or more files or tracks
    • list of reference -- the file(s)/tracks(s) to convert
    • Result: track -- reference to converted track(s)
  • fast_forward -- skip forward in a playing track
  • next_track -- advance to the next track in the current playlist
  • pause -- pause playback
  • play -- play the current track or the specified track or file.
    • [reference] -- item to play
    • [once boolean] -- If true, play this track once and then stop.
  • playpause -- toggle the playing/paused state of the current track
  • previous_track -- return to the previous track in the current playlist
  • refresh -- update file track information from the current information in the track’s file
    • file_track -- the file track to update
  • resume -- disable fast forward/rewind and resume playback, if playing.
  • reveal -- reveal and select a track or playlist
    • item -- the item to reveal
  • rewind -- skip backwards in a playing track
  • search -- search a playlist for tracks matching the search string. Identical to entering search text in the Search field in iTunes.
    • playlist -- the playlist to search
    • for unicode_text -- the search text
    • [only :albums / :all / :artists / :composers / :displayed / :songs] -- area to search (default is all)
    • Result: track -- reference to found track(s)
  • stop -- stop playback
  • update -- update the specified iPod
    • [source] -- the iPod to update
  • eject -- eject the specified iPod
    • [source] -- the iPod to eject
  • subscribe -- subscribe to a podcast feed
    • string -- the URL of the feed to subscribe to
  • updateAllPodcasts -- update all subscribed podcast feeds
  • updatePodcast -- update podcast feed
  • download -- download podcast episode
    • URL_track

Classes

  • application -- The application program
    • Properties:
      • current_encoder encoder -- the currently selected encoder (MP3, AIFF, WAV, etc.)
      • current_EQ_preset EQ_preset -- the currently selected equalizer preset
      • current_playlist (r/o) playlist -- the playlist containing the currently targeted track
      • current_stream_title (r/o) unicode_text -- the name of the current song in the playing stream (provided by streaming server)
      • current_stream_URL (r/o) unicode_text -- the URL of the playing stream or streaming web site (provided by streaming server)
      • current_track (r/o) track -- the current targeted track
      • current_visual visual -- the currently selected visual plug-in
      • EQ_enabled boolean -- is the equalizer enabled?
      • fixed_indexing boolean -- true if all AppleScript track indices should be independent of the play order of the owning playlist.
      • frontmost boolean -- is iTunes the frontmost application?
      • full_screen boolean -- are visuals displayed using the entire screen?
      • name (r/o) unicode_text -- the name of the application
      • mute boolean -- has the sound output been muted?
      • player_position integer -- the player’s position within the currently playing track in seconds.
      • player_state (r/o) :stopped / :playing / :paused / :fast_forwarding / :rewinding -- is iTunes stopped, paused, or playing?
      • selection (r/o) reference -- the selection visible to the user
      • sound_volume integer -- the sound output volume (0 = minimum, 100 = maximum)
      • version (r/o) version -- the version of iTunes
      • visuals_enabled boolean -- are visuals currently being displayed?
      • visual_size :small / :medium / :large -- the size of the displayed visual
    • Elements:
      • browser_windows -- by index, name, id
      • encoders -- by index, name, id
      • EQ_presets -- by index, name, id
      • EQ_windows -- by index, name, id
      • playlist_windows -- by index, name, id
      • sources -- by index, name, id
      • visuals -- by index, name, id
      • windows -- by index, name, id
  • artwork -- a piece of art within a track
    • Plural name:
      • artworks
    • Inherits from:

    • Properties:
      • data_ PICT_picture -- data for this artwork, in the form of a picture
      • description unicode_text -- description of artwork as a string
      • downloaded (r/o) boolean -- was this artwork downloaded by iTunes?
      • format (r/o) type_class -- the data format for this piece of artwork
      • kind integer -- kind or purpose of this piece of artwork
      • raw_data AEType.new("tdta") -- data for this artwork, in original format

      • disc_number integer -- the index of this CD disc in the source album
      • genre unicode_text -- the genre of the CD
      • year integer -- the year the album was recorded/released
    • Elements:

      • view playlist -- the playlist currently displayed in the window

    • Plural name:

  • EQ_preset -- equalizer preset configuration
    • Plural name:
      • EQ_presets
    • Inherits from:
      • item
    • Properties:
      • band_1 short_float -- the equalizer 32 Hz band level (-12.0 dB to +12.0 dB)
      • band_2 short_float -- the equalizer 64 Hz band level (-12.0 dB to +12.0 dB)
      • band_3 short_float -- the equalizer 125 Hz band level (-12.0 dB to +12.0 dB)
      • band_4 short_float -- the equalizer 250 Hz band level (-12.0 dB to +12.0 dB)
      • band_5 short_float -- the equalizer 500 Hz band level (-12.0 dB to +12.0 dB)
      • band_6 short_float -- the equalizer 1 kHz band level (-12.0 dB to +12.0 dB)
      • band_7 short_float -- the equalizer 2 kHz band level (-12.0 dB to +12.0 dB)
      • band_8 short_float -- the equalizer 4 kHz band level (-12.0 dB to +12.0 dB)
      • band_9 short_float -- the equalizer 8 kHz band level (-12.0 dB to +12.0 dB)
      • band_10 short_float -- the equalizer 16 kHz band level (-12.0 dB to +12.0 dB)
      • modifiable (r/o) boolean -- can this preset be modified?
      • preamp short_float -- the equalizer preamp level (-12.0 dB to +12.0 dB)
      • update_tracks boolean -- should tracks which refer to this preset be updated when the preset is renamed or deleted?

      • special_kind (r/o) :none / :Books / :folder / :Genius / :iTunes_U / :Library / :Movies / :Music / :Party_Shuffle / :Podcasts / :Purchased_Music / :TV_Shows -- special playlist kind

      • visible (r/o) boolean -- is this playlist visible in the Source list?
    • Elements:
      • tracks -- by index, name, id

    • Plural name:

    • Inherits from:

      • free_space (r/o) double_integer -- the free space on the source if it has a fixed size

    • Elements:
      • audio_CD_playlists -- by index, name, id
      • device_playlists -- by index, name, id

  • track -- playable audio source
    • Plural name:
      • tracks
    • Inherits from:
      • item
    • Inherited by:
      • audio_CD_track
      • device_track
      • file_track
      • shared_track
      • URL_track
    • Properties:
      • album unicode_text -- the album name of the track
      • album_artist unicode_text -- the album artist of the track
      • album_rating integer -- the rating of the album for this track (0 to 100)
      • album_rating_kind (r/o) :user / :computed -- the rating kind of the album rating for this track
      • artist unicode_text -- the artist/source of the track
      • bit_rate (r/o) integer -- the bit rate of the track (in kbps)
      • bookmark short_float -- the bookmark time of the track in seconds
      • bookmarkable boolean -- is the playback position for this track remembered?
      • bpm integer -- the tempo of this track in beats per minute
      • category unicode_text -- the category of the track
      • comment unicode_text -- freeform notes about the track
      • compilation boolean -- is this track from a compilation album?
      • composer unicode_text -- the composer of the track
      • database_ID (r/o) integer -- the common, unique ID for this track. If two tracks in different playlists have the same database ID, they are sharing the same data.
      • date_added (r/o) date -- the date the track was added to the playlist
      • description unicode_text -- the description of the track
      • disc_count integer -- the total number of discs in the source album
      • disc_number integer -- the index of the disc containing this track on the source album
      • duration (r/o) short_float -- the length of the track in seconds
      • enabled boolean -- is this track checked for playback?
      • episode_ID unicode_text -- the episode ID of the track
      • episode_number integer -- the episode number of the track
      • EQ unicode_text -- the name of the EQ preset of the track
      • finish short_float -- the stop time of the track in seconds
      • gapless boolean -- is this track from a gapless album?
      • genre unicode_text -- the music/audio genre (category) of the track
      • grouping unicode_text -- the grouping (piece) of the track. Generally used to denote movements within a classical work.
      • kind (r/o) unicode_text -- a text description of the track
      • long_description unicode_text
      • lyrics unicode_text -- the lyrics of the track
      • modification_date (r/o) date -- the modification date of the content of this track
      • played_count integer -- number of times this track has been played
      • played_date date -- the date and time this track was last played
      • podcast (r/o) boolean -- is this track a podcast episode?
      • rating integer -- the rating of this track (0 to 100)
      • rating_kind (r/o) :user / :computed -- the rating kind of this track
      • release_date (r/o) date -- the release date of this track
      • sample_rate (r/o) integer -- the sample rate of the track (in Hz)
      • season_number integer -- the season number of the track
      • shufflable boolean -- is this track included when shuffling?
      • skipped_count integer -- number of times this track has been skipped
      • skipped_date date -- the date and time this track was last skipped
      • show unicode_text -- the show name of the track
      • sort_album unicode_text -- override string to use for the track when sorting by album
      • sort_artist unicode_text -- override string to use for the track when sorting by artist
      • sort_album_artist unicode_text -- override string to use for the track when sorting by album artist
      • sort_name unicode_text -- override string to use for the track when sorting by name
      • sort_composer unicode_text -- override string to use for the track when sorting by composer
      • sort_show unicode_text -- override string to use for the track when sorting by show name
      • size (r/o) integer -- the size of the track (in bytes)
      • start short_float -- the start time of the track in seconds
      • time (r/o) unicode_text -- the length of the track in MM:SS format
      • track_count integer -- the total number of tracks on the source album
      • track_number integer -- the index of the track on the source album
      • unplayed boolean -- is this track unplayed?
      • video_kind :none / :movie / :music_video / :TV_show -- kind of video track
      • volume_adjustment integer -- relative volume adjustment of the track (-100% to 100%)
      • year integer -- the year the track was recorded/released
    • Elements:
      • artworks -- by index
  • URL_track -- a track representing a network stream
    • Plural name:
      • URL_tracks
    • Inherits from:
      • track
    • Properties:
      • address unicode_text -- the URL for this track
  • user_playlist -- custom playlists created by the user
    • Plural name:
      • user_playlists
    • Inherits from:
      • playlist
    • Inherited by:

    • Properties:

      • smart (r/o) boolean -- is this a Smart Playlist?
    • Elements:
      • file_tracks -- by index, name, id
      • URL_tracks -- by index, name, id
      • shared_tracks -- by index, name, range, relative, test
  • visual -- a visual plug-in
    • Plural name:
      • visuals

    • Plural name:
